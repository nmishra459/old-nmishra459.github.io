Catch 21: Dice

This was my final project for CS 50! My partner and I implemented a web application to host a two-player dice game progammed with JavaScript. The game also features a username/password account system, a personalized game statistics page, and a global leaderboard for users to see how they stack up against other, all implemented with SQL and Python. For the front-end, we used HTML5/CSS, as well as Bootstrap 4. We've deployed the website on Heroku, so anyone can make an account, play with friends, and appear on the leaderboard!

Keep on Trucking

In March 2020, I teamed up with 4 of my high school friends to compete in the MathWorks Math Modeling Challenge. Together, we researched the U.S. electric truck market, developed mathematical models (Python) to predict electric truck usage in the next two decades, and determined ideal economic/environmental zones for U.S.-based charging station infrastructure. Our final paper was placed in the top 19% of 760 submissions.

Asteroid Ambush!

For my final project for AP Computer Science A, I programmed a 2D space fighter game that has users destroy asteroids and collect coins. The game features a timer, scoreboard, and life counter. To spice up gameplay, I added four different upgrade paths for the users to take, which can either grant lives, randomly destroy asteroids, or improve the ray-gun mechanism. The spirtes were taken from open source locations on the internet, and the logic was implemented entirely in Java.
